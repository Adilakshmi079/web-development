HTML stands for Hyper Text Markup Language, is the standard language for creating web pages. It provides the structure and content of a webpage, telling web browsers how to display text, images, and other media. HTML uses a series of elements, denoted by tags, to define different parts of a webpage, like headings, paragraphs, and links.

CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media CSS saves a lot of work. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files
